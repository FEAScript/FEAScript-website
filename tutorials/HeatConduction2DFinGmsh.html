This example demonstrates solving a steady-state heat transfer problem in a 2D rectangular domain using a mesh generated by GMSH. GMSH is a powerful mesh generation tool that can create complex geometries and meshes for finite element analysis.

Introduction

This example uses a GMSH-generated mesh to solve a typical cooling fin problem. The objective is to model heat conduction and understand temperature distribution under specific boundary conditions.
